## Getting Started

### Clone the repository

```bash
git clone https://github.com/Muhammed-Anas-A/HtmlCssUsingTailwind.git
```

### Open the project

Navigate to any project folder and open the `index.html` file in your browser.

If Tailwind is installed using npm:

```bash
npm install
npm run dev
```
